什么是外部样式表？

外部样式表是将 CSS 代码写在独立的 .css 文件中， 然后通过 <link> 标签引入到 HTML 文件中。

优点：

  • ✅ 样式和结构完全分离，代码更清晰
  • ✅ 一个 CSS 文件可以被多个 HTML 页面复用
  • ✅ 修改样式只需要改一个文件
  • ✅ 浏览器会缓存 CSS 文件，提高加载速度
  • ✅ 便于团队协作和维护

如何使用外部样式表？

第1步：创建 CSS 文件

/* external-style.css */
body {
    font-family: Arial, sans-serif;
    background-color: #f5f5f5;
}

.header {
    background-color: #667eea;
    color: white;
    padding: 30px;
}

第2步：在 HTML 中引入

<head>
    <link rel="stylesheet" href="external-style.css">
</head>

本页面的所有样式（头部、导航、卡片、按钮等）都定义在 external-style.css 文件中。

  2. 修改某个样式（比如 .header 的背景色）
  3. 刷新浏览器，查看效果
  4. 你会发现修改非常方便！

💡 最佳实践

  • ✓ 将所有样式放在外部 CSS 文件中
  • ✓ 使用有意义的类名（如 .header, .nav, .card）
  • ✓ 按功能模块组织 CSS 代码
  • ✓ 添加注释说明样式的用途
  • ✓ 大型项目可以分多个 CSS 文件（header.css, footer.css）
