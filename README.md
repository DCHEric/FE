# 前端基础知识学习指南

欢迎来到前端开发的世界！本项目将通过实际代码示例帮助你学习前端三大核心技术。

## 📚 学习路径

### 1️⃣ HTML 基础 (01-html-basics.html)
HTML 是网页的骨架，定义了内容的结构。

**学习内容：**
- 基本标签：标题、段落、链接、图片
- 列表：有序列表、无序列表
- 表格：创建数据表格
- 表单：输入框、按钮、下拉菜单
- 语义化标签：header, nav, main, article, footer

**如何学习：**
1. 在浏览器中打开 `01-html-basics.html`
2. 查看源代码，理解每个标签的作用
3. 尝试修改内容，看看效果如何变化

### 2️⃣ CSS 基础 (02-css-basics.html)
CSS 负责网页的外观和布局，让网页变得美观。

**学习内容：**
- 选择器：类、ID、元素选择器
- 盒模型：padding, border, margin
- 文字样式：字体、颜色、对齐
- 布局：Flexbox、Grid、定位
- 动画：过渡效果、关键帧动画
- 响应式设计：媒体查询

**如何学习：**
1. 在浏览器中打开 `02-css-basics.html`
2. 使用浏览器开发者工具 (F12) 实时修改样式
3. 尝试改变颜色、大小、布局

### 3️⃣ JavaScript 基础 (03-javascript-basics.html)
JavaScript 让网页具有交互性和动态功能。

**学习内容：**
- 变量和数据类型
- 条件语句和循环
- 数组和对象
- 函数
- DOM 操作：修改网页元素
- 事件处理：响应用户操作
- 现代 JavaScript (ES6+)

**如何学习：**
1. 在浏览器中打开 `03-javascript-basics.html`
2. 点击页面上的按钮，查看交互效果
3. 打开浏览器控制台 (F12 → Console) 查看输出
4. 修改代码，尝试创建自己的功能

## 🚀 快速开始

### 方法一：直接在浏览器打开
1. 在文件管理器中找到 HTML 文件
2. 双击文件，会在默认浏览器中打开
3. 开始学习！

### 方法二：使用 VS Code Live Server
1. 在 VS Code 中安装 "Live Server" 扩展
2. 右键点击 HTML 文件
3. 选择 "Open with Live Server"
4. 文件会在浏览器中自动打开，并支持热更新

## 💡 学习建议

1. **按顺序学习**: HTML → CSS → JavaScript
2. **动手实践**: 不要只看代码，要修改和实验
3. **使用开发者工具**: 按 F12 打开，可以：
   - 查看和修改 HTML 结构
   - 实时调试 CSS 样式
   - 在 Console 中运行 JavaScript 代码
4. **做笔记**: 记录重要概念和常用代码
5. **创建项目**: 学完基础后，尝试创建自己的网页

## 🔧 开发者工具快捷键

- **F12** 或 **Ctrl+Shift+I** (Windows/Linux) / **Cmd+Option+I** (Mac): 打开开发者工具
- **Ctrl+Shift+C** / **Cmd+Shift+C**: 元素选择器
- **Ctrl+Shift+J** / **Cmd+Option+J**: 打开控制台
