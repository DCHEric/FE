CSS 学习小结

  • 选择器：元素选择器、类选择器(.class)、ID选择器(#id)
  • 盒模型：content、padding、border、margin
  • 布局：display、position、flexbox、grid
  • 样式：颜色、字体、背景、边框
  • 动画：transition、animation
  • 响应式：媒体查询 @media
