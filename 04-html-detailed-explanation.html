📋 第一课：HTML 文档的基本结构

💡 核心概念

每个 HTML 文档都像一本书，有固定的结构：

  • 声明部分：告诉浏览器这是 HTML5 文档
  • 头部 (head)：包含页面的元信息（用户看不到）
  • 身体 (body)：包含页面的实际内容（用户能看到）
<!DOCTYPE html> ← 文档类型声明，必须是第一行 <html lang="zh-CN"> ← 根元素，lang 指定语言 <head> ← 头部开始 <meta charset="UTF-8"> ← 字符编码，支持中文 <title>网页标题</title> ← 浏览器标签页显示的标题 </head> ← 头部结束 <body> ← 身体开始 <h1>这是标题</h1> ← 网页内容 <p>这是段落</p> ← 网页内容 </body> ← 身体结束 </html> ← 根元素结束

🏷️ 第二课：什么是标签？

💡 标签的组成

HTML 使用标签 (Tags)来标记内容。大多数标签成对出现：

标签的解剖：

<p>这是段落内容</p>
 ↑                    ↑
开始标签           结束标签
(有 < >)        (有 </ >)

标签的三种形式：

  1. 双标签（成对）：<p>内容</p> - 有开始和结束
  2. 单标签（自闭合）：<br> 或 <img /> - 只有一个标签
  3. 带属性的标签：<a href="网址">链接</a> - 标签可以有属性

📝 第三课：常用标签详解

1️⃣ 标题标签 (h1-h6)

作用：定义标题，从 h1 到 h6，重要性递减

h1 - 最重要的标题（每页只用一次）

h2 - 二级标题

h3 - 三级标题

h4 - 四级标题

h5 - 五级标题
h6 - 六级标题
<h1>网站的主标题</h1> <h2>章节标题</h2> <h3>小节标题</h3>
💡 使用技巧：
  • h1 最重要，通常每页只有一个（比如网站名称或文章标题）
  • 按层级使用，不要跳级（不要从 h1 直接跳到 h3）
  • 不要仅仅为了大字体而使用标题标签（用 CSS 控制样式）

2️⃣ 段落和文本标签

标签 作用 示例 效果
<p> 段落 <p>这是一段文字</p>

这是一段文字
<br> 换行 第一行<br>第二行 第一行
第二行
<strong> 强调（粗体） <strong>重要</strong> 重要
<em> 强调（斜体） <em>强调</em> 强调
<mark> 高亮 <mark>标记</mark> 标记
<small> 小字 <small>小字</small> 小字

3️⃣ 链接标签 <a>

作用：创建超链接，让用户可以跳转到其他页面

<!-- 基本链接 --> <a href="https://www.baidu.com">去百度</a> <!-- 新窗口打开 --> <a href="https://www.baidu.com" target="_blank">在新窗口打开百度</a> <!-- 链接到页面中的某个位置 --> <a href="#section1">跳转到第一节</a> <!-- 邮件链接 --> <a href="mailto:example@email.com">发送邮件</a>

链接示例：

MDN Web 文档（新窗口）

返回页面顶部

4️⃣ 图片标签 <img>

作用：在网页中显示图片（单标签）

<img src="图片路径" alt="图片描述" width="宽度" height="高度"> <!-- 实际例子 --> <img src="logo.png" alt="公司Logo" width="200">
💡 重要属性：
  • src（必需）：图片的路径（URL 或本地路径）
  • alt（必需）：图片的替代文字，当图片无法显示时显示这段文字
  • width/height（可选）：设置图片宽高

5️⃣ 列表标签

无序列表 (ul)

<ul> <li>苹果</li> <li>香蕉</li> <li>橙子</li> </ul>

效果：

  • 苹果
  • 香蕉
  • 橙子

有序列表 (ol)

<ol> <li>第一步</li> <li>第二步</li> <li>第三步</li> </ol>

效果：

  1. 第一步
  2. 第二步
  3. 第三步

6️⃣ 容器标签：div 和 span

这两个是最常用的容器标签，用于组织和布局内容：

标签 类型 特点 用途
<div> 块级元素 独占一行 页面布局、分组内容
<span> 行内元素 不换行 修饰文本中的一部分

🎯 第四课：语义化标签 (HTML5)

💡 什么是语义化？

语义化标签用标签名就能看出内容的含义，而不是所有地方都用 div。

好处：

  • ✅ 代码更易读、易维护
  • ✅ 有利于 SEO（搜索引擎优化）
  • ✅ 辅助功能更好（帮助视障人士）

典型页面结构：

<header> - 页面头部（Logo、导航）
<nav> - 导航栏
<main> - 主要内容区
<article> - 文章
<section> - 章节
<aside>
侧边栏
<footer> - 页面底部（版权、链接）
标签 含义 适用场景
<header> 头部区域 网站头部、文章标题区
<nav> 导航链接区 菜单、面包屑导航
<main> 主要内容 页面的主体内容（每页只用一次）
<article> 独立文章 博客文章、新闻报道
<section> 章节/区块 内容分组、主题分区
<aside> 侧边内容 侧边栏、相关链接
<footer> 底部区域 版权信息、联系方式

📊 第六课：表格

表格用于展示结构化的数据，由行和列组成。

<table> <thead> ← 表头 <tr> ← 行 <th>姓名</th> ← 表头单元格 <th>年龄</th> </tr> </thead> <tbody> ← 表格主体 <tr> <td>张三</td> ← 普通单元格 <td>25</td> </tr> </tbody> </table>

表格示例：

学号 姓名 成绩 等级

002 李四 88 良好
003 王五 76 中等
